# Jarvis
Iron man Jarvis Chatbot Project

## Installation
```
conda env create --file environment.yaml
```
## How to use
### Train
```
python train.py
```
### Inference
```
python inference.py
```

## Example
```
Input: 전 코딩을 너무 못해요
Output: 정말 힘드시겠어요 .  하지만 너무 자책하지 마세요 .
Input: 할줄 아는게 아무 것도 없어요

Input: 킥복싱을 잘하고 싶어요
```
